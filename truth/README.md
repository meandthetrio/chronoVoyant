# truth/ — The Truth-Avoidance Detector

> "Whether a line is true is subjective; whether a line is *specific* is
> objective, and vagueness is almost always where the avoidance is
> hiding." — The Bible

This is the subtlest of the four strengths, so it gets the sharpest
boundary. Claude cannot know if a line is true — only you can. What
Claude can detect, mechanically, is the *tell*: abstraction. A lyric that
states a feeling instead of showing a scene is the songwriting
equivalent of a rationalized trade.

The chalupa wrapper on the passenger seat beats "I feel so empty inside"
every time — not because specificity is a style, but because vagueness
is where the avoidance hides.

Contents:

- `specificity-test.md` — the test Claude runs on flagged lines

The output of this folder is always a question, never a rewrite. Claude
flags the line and asks "what actually happened?" — and then you answer
it, because the answer is in your life, not in any model.
