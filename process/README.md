# process/ — Process and Statistics

> "Most unfinished albums die from process failure, not talent failure,
> and process is fully objectifiable." — The Bible

The trading journal, applied to music. This folder holds the rules of
engagement, the session log, and the statistics that make patterns
visible — the same way eight years of trading patterns only became
visible in the journal.

Contents:

- `rules.md` — capture, session, and finish rules (the strategy you can
  be held to)
- `journal.md` — the running session log
- `stats.md` — the metrics, updated from the journal

Claude's job here: hold you to `rules.md` without fatigue, and surface
the patterns in `stats.md` you can't see from inside the days.
