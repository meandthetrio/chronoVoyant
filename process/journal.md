# Session Journal

Append-only. Newest at the bottom. Two lines minimum per session:

```
## <date> <time>
Intention: <one line, written at session start>
Result: <one line, written at session end — what happened, status changes>
```

Optional third line — `Reach: <note>` — if The Reach showed up during
the session. That data belongs in this project twice over.
