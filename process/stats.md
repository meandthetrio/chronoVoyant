# Stats

Computed from `journal.md`, the git history, and the folder contents.
Updated by Claude on request ("update the stats"). Numbers only — the
interpretation happens in conversation.

## Tracked metrics

- Sessions per week (from journal entries)
- Fragments captured (from `lyrics/fragments.md`)
- Fragment → song conversion rate (fragments that appear in a song draft)
- Songs by status: sketch / drafting / parked / finished
- Median days in "drafting" before finish or park
- Intention-kept rate: sessions whose Result matches their Intention
- Theme recurrence: which `ideas/` themes appear in journal entries vs.
  in actual drafts (the gap between what you circle and what you write)
